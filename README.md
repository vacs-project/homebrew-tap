# vacs-project Tap

Homebrew tap for vacs - the VATSIM ATC Communication System.

## How do I install vacs using brew?

`brew install --cask vacs-project/tap/vacs`

Or `brew tap vacs-project/tap` and then `brew install --cask vacs`.

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
